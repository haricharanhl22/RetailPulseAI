# RetailPulse AI

RetailPulse AI is an intelligent retail analytics platform built using Python, Streamlit, Plotly, Scikit-learn, and Prophet.

The platform provides:
- Interactive business dashboards
- Sales forecasting
- AI-powered anomaly detection
- Automated business insights
- Regional filtering and KPI analysis

## Features

### Business Intelligence Dashboard
- Revenue analysis
- Profitability tracking
- Category performance
- Regional insights

### AI-Powered Analytics
- Anomaly detection using Isolation Forest
- Predictive sales forecasting using Prophet

### Interactive UI
- Streamlit dashboard
- Plotly visualizations
- Dynamic filtering

## Tech Stack

- Python
- Pandas
- Plotly
- Streamlit
- Scikit-learn
- Prophet
